# Kalshi Temperature Market Scanner

Real-time dashboard for Kalshi daily high-temperature prediction markets.

**Live site:** https://yw666.github.io/kalshi_weather_betting

## How it works

- **GitHub Actions** runs `fetch.py` every 15 minutes, saves results to `data.json`, and pushes to the repo
- **GitHub Pages** serves `index.html` which reads `data.json` directly — no server required
- The page auto-refreshes every 15 minutes to pick up the latest `data.json`

## Setup

1. Go to **Settings → Pages** → set source to `main` branch, `/ (root)` folder → Save
2. Go to **Settings → Actions → General** → make sure "Read and write permissions" is enabled under Workflow permissions
3. Trigger the first run manually: **Actions → Fetch Kalshi Data & Deploy → Run workflow**

The site will be live at `https://yw666.github.io/kalshi_weather_betting`

## Files

| File | Purpose |
|------|---------|
| `fetch.py` | Python script that fetches Kalshi API and saves `data.json` |
| `index.html` | Static dashboard that reads `data.json` |
| `data.json` | Auto-generated by Actions, contains latest market data |
| `.github/workflows/fetch.yml` | Actions workflow — runs every 15 min |
